Sets one of the streams (input, output, or error) to a `stream` type resource.

public static setStream(string $whichStream, resource $stream) : void

Valid $whichStream values are:

  • 'in' (default: STDIN)
  • 'out' (default: STDOUT)
  • 'err' (default: STDERR)

Any custom streams will be closed for you on shutdown, so please don't close stream resources used with this method.
