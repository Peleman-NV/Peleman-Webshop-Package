empty_()

Site_Command.php : 250

Empties a site of its content (posts, comments, terms, and meta).

public empty_(mixed $args, mixed $assoc_args) : mixed

Truncates posts, comments, and terms tables to empty a site of its content. Doesn't affect site configuration (options) or users.

If running a persistent object cache, make sure to flush the cache after emptying the site, as the cache values will be invalid otherwise.

To also empty custom database tables, you'll need to hook into command execution:

WP_CLI::add_hook( 'after_invoke:site empty', function(){
    global $wpdb;
    foreach( array( 'p2p', 'p2pmeta' ) as $table ) {
        $table = $wpdb->$table;
        $wpdb->query( "TRUNCATE $table" );
    }
});

OPTIONS

[--uploads] : Also delete all files in the site's uploads directory.

[--yes] : Proceed to empty the site without a confirmation prompt.

EXAMPLES

$ wp site empty
Are you sure you want to empty the site at http://www.example.com of all posts, links, comments, and terms? [y/n] y
Success: The site at 'http://www.example.com' was emptied.

list_()

Site_Command.php : 570

Lists all sites in a multisite installation.

public list_(mixed $args, mixed $assoc_args) : mixed

OPTIONS

[--network=] : The network to which the sites belong.

[--=] : Filter by one or more fields (see "Available Fields" section). However, 'url' isn't an available filter, because it's created from domain + path.

[--site__in=] : Only list the sites with these blog_id values (comma-separated).

[--field=] : Prints the value of a single field for each site.

[--fields=] : Comma-separated list of fields to show.

[--format=] : Render output in a particular format.

default: table options:

  • table
  • csv
  • count
  • ids
  • json
  • yaml

AVAILABLE FIELDS

These fields will be displayed by default for each site:

  • blog_id
  • url
  • last_updated
  • registered

These fields are optionally available:

  • site_id
  • domain
  • path
  • public
  • archived
  • mature
  • spam
  • deleted
  • lang_id

# Output a simple list of site URLs
$ wp site list --field=url
http://www.example.com/
http://www.example.com/subdir/
